# Malformed JSON sample note

<section lang="fa" dir="rtl">

این فایل عمداً JSON نیست. هدف آن توضیح سناریوی `invalid` برای کاربر شخصی است.

اگر کاربر محتوای JSON خراب یا فایل غیر JSON را به Project Gate بدهد، نتیجه امن باید `invalid` باشد و نباید سیستم خودش ساختار را حدس بزند یا اصلاح کند.

این فایل synthetic است و شواهد واقعی EV4 محسوب نمی‌شود.
